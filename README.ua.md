# Weather App (Додаток погоди)

Сучасний адаптивний додаток для прогнозу погоди, побудований на Vue 3, Vite та Tailwind CSS. Додаток дозволяє користувачам шукати міста та переглядати актуальний прогноз погоди.

## 🚀 Особливості

- **Пошук міст:** Пошук будь-якого міста світу за допомогою інтеграції геолокації.
- **Прогноз погоди:** Детальна інформація про погоду та прогнози.
- **Сучасний інтерфейс:** Чистий, адаптивний дизайн з використанням Tailwind CSS та ефектів glassmorphism.
- **Управління станом:** Ефективна обробка даних за допомогою Pinia.
- **TypeScript:** Повна типізація для кращої розробки та надійності.

## 🛠️ Технологічний стек

- **Фреймворк:** [Vue 3](https://vuejs.org/) (Composition API)
- **Інструмент збірки:** [Vite](https://vitejs.dev/)
- **Управління станом:** [Pinia](https://pinia.vuejs.org/)
- **Стилізація:** [Tailwind CSS](https://tailwindcss.com/)
- **HTTP-клієнт:** [ofetch](https://github.com/unjs/ofetch)
- **Робота з датами:** [dayjs](https://day.js.org/)
- **Мова:** [TypeScript](https://www.typescriptlang.org/)

## 🏁 Початок роботи

### Попередні вимоги

- Node.js (рекомендовано версію 16 або вище)
- pnpm (бажано) або npm/yarn

### Встановлення

1. Клонуйте репозиторій:
   ```bash
   git clone <repository-url>
   cd weather-app
   ```

2. Встановіть залежності:
   ```bash
   pnpm install
   ```

### Розробка

Запустіть сервер для розробки:
```bash
pnpm run dev
```

### Продакшн

Зберіть проект для продакшну:
```bash
pnpm run build
```

Перегляньте локально зібрану версію:
```bash
pnpm run preview
```

## 🧪 Скрипти

- `pnpm run dev` - запуск сервера для розробки.
- `pnpm run build` - збірка для продакшну.
- `pnpm run preview` - перегляд зібраної версії.
- `pnpm run lint` - запуск ESLint для виправлення стилю коду.
- `pnpm run type-check` - запуск перевірки типів TypeScript.
