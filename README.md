# Weather App

A modern, responsive weather forecast application built with Vue 3, Vite, and Tailwind CSS. This app allows users to search for cities and view current weather forecasts.

## 🚀 Features

- **City Search:** Search for any city worldwide using geolocation integration.
- **Weather Forecast:** Detailed weather information and forecasts.
- **Modern UI:** Clean, responsive design using Tailwind CSS with glassmorphism effects.
- **State Management:** Efficient data handling with Pinia.
- **TypeScript:** Fully typed for better developer experience and reliability.

## 🛠️ Tech Stack

- **Framework:** [Vue 3](https://vuejs.org/) (Composition API)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **State Management:** [Pinia](https://pinia.vuejs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **HTTP Client:** [ofetch](https://github.com/unjs/ofetch)
- **Date Utilities:** [dayjs](https://day.js.org/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)

## 🏁 Getting Started

### Prerequisites

- Node.js (version 16 or higher recommended)
- pnpm (preferred) or npm/yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd weather-app
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

### Development

Run the development server:
```bash
pnpm run dev
```

### Production

Build for production:
```bash
pnpm run build
```

Preview production build locally:
```bash
pnpm run preview
```

## 🧪 Scripts

- `pnpm run dev` - Start development server.
- `pnpm run build` - Build for production.
- `pnpm run preview` - Preview production build.
- `pnpm run lint` - Run ESLint to fix code style.
- `pnpm run type-check` - Run TypeScript type checking.
